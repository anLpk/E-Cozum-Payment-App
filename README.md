# E-Çözüm Payment APP

This project is a Payment App and is built for E-Çözüm & Patika Bootcamp.

## Technologies used in this project

- React
- Typescript
- Redux-Toolkit
- Redux-Thunk
- React-Router
- Cleave JS
- Validator
- React Icons
- Axios
- Chakra-UI
